# Link CLI

```
NAME:
   formatter - A new cli application

USAGE:
   formatter [global options] command [command options] [arguments...]

VERSION:
   1.0.0

COMMANDS:
     list
     destroy
     get
     try-get-lock
     add
     help, h       Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --host value                Host to contact (default: "127.0.0.1:1313")
   --user value, -u value      Username for basic auth
   --password value, -p value  Password for basic auth
   --help, -h                  show help
   --version, -v               print the version
```
